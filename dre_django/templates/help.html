Ajuda

Procurar

Os documentos, extraídos do Diário da República Electrónico são indexados neste site. Para realizar uma busca basta usar termos que sejam adequados. Realizaremos então uma busca sobre todo o texto disponível da legislação assim como sobre os metadados disponibilizados no site oficial do DR Electrónico (dre.pt).

Para além disto, o nosso motor de busca suporta a utilização de modificadores que tornam as buscas mais eficientes. Por exemplo para se procurar apenas por leis, pode-se acrescentar: tipo:lei. Da mesma forma, se um determinado documento que procuramos é de um ano conhecido, podemos modificar a busca utilizando: ano:1989.

Os modificadores suportados, neste momento, são:

  • data - aceita uma data no formato AAAAMMDD;
  • ano - ano do documento, no formato AAAA;
  • mês - mês do documento, no formato MM ou M, sem zeros à esquerda;
  • dia - dia do documento, no formato DD ou D, sem zeros à esquerda;
  • tipo - tipo de documento, a seguir listamos os tipos de documentos suportados;
  • número - número do documento, convém ser o número exacto. Notar que os documentos antes do ano dois mil identificam o ano com dois caracteres, a partir do ano dois mil, utilizam quatro dígitos. Isto é apenas uma regra que muitas vezes não é seguida. O número de documento utilizado com este modificador deve ser exacto;
  • quem - entidade emitente;
  • série - série do Diário da República (1,2 ou 3);
  • dr - número do Diário da República;

Exemplos

Procurar leis sobre os caminhos de ferro:

"caminhos de ferro" tipo:lei

Procurar portarias de Maio de 1989:

ano:1989 mês:5 tipo:portaria

Notas

Os utilizadores registados podem criar notas para cada documento. Essas notas podem conter até vinte mil caracteres cada uma. Para além de texto simples podem ser formatadas utilizando a linguagem 'markdown'. Esta é uma linguagem extremamente simples. Por exemplo se quiser realçar com negrito uma dada palavra, apenas tem de envolver essa palavra em '**', a frase seguinte: 

  Este é um **texto** de exemplo.
Ficaria:

Este é um texto de exemplo.
